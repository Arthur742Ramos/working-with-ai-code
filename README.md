# Working with AI as a Real Teammate

Code examples and resources for the Manning book *Working with AI as a Real Teammate* by Arthur Freitas Ramos.

## About the Book

This book teaches software engineers to integrate AI into their workflows using the 3C Loop (Contract, Conversation, Checks), a mental model for reliable AI collaboration. It is tool-agnostic: the patterns work with any AI assistant.

## Requirements

- Python 3.10+
- Dependencies per chapter listed in each folder's `requirements.txt`

## Setup

```bash
# Clone the repository
git clone https://github.com/Arthur742Ramos/working-with-ai-code.git
cd working-with-ai-code

# Create a virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies for a specific chapter
pip install -r ch02/requirements.txt
```
